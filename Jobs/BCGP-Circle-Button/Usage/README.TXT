Usage:

1. Add `BCGPCircleButton.h` and `BCGPCircleButton.cpp` to project.
2. Drag & Drop a push button into a dialog and adjust it to be a square.
3. Add a variable of this button with `CBCGPCircleButton` class.
4. Add its event handlers.
5. Finish!!

Note: Open the `sample circle button - code changes.html` to see the changes of a sample.
