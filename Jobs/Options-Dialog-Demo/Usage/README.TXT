Usage:

1. Add all h/cpp files in the `Codes` folder to your project.
2. Basing on the tracked modifications in the `Options-Dialog-Demo.html` file, make corresponding changes in your project.
	2.1. Important: The changes in the `resource.h` and `.rc` file is apart of the feature. It contains UI (Dialog, Control IDs, etc).
	2.2. The other changes are applying to use this feature.
3. Finished !!.
